# Bubble Slider
Bubble Slider is a simple slider widget that shows position in a popup bubble with smooth animation. It uses Fluid Slider library by Ramotion.

---

## Properties
---

> ![Height](./images/height.png)

### Height
Set height for slider. You need to assign this value before using the create block. Else it will not work.

---

> ![BarColor](./images/bar_color.png)

### Bar Color
Set background color of bar.

---

> ![BubbleColor](./images/bubble_color.png)

### Bubble Color
Color of circular “bubble” inside bar.

---

> ![BarTextColor](./images/bar_text_color.png)

### Bar Text Color
Color of start and end texts of slider.

---

> ![BubbleTextColor](./images/bubble_text_color.png)

### Bubble Text Color
Color of text inside “bubble”.

---

> ![StartText](./images/start_text.png)

### Start Text
Start (left) text of slider.

---

> ![EndText](./images/end_text.png)

### End Text
End (right) text of slider.

---

> ![FontSize](./images/font_size.png)

### Font Size

---

> ![Postion](./images/position.png)

> ![RiseDuration](./images/rise_duration.png)
